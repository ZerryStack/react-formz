### useFormSubmission

▸ **useFormSubmission**<`Values`\>(`id`, `onSubmit?`, `resetOnSubmit?`): (`e`: `FormEvent`<`HTMLFormElement`\>) => `Promise`<`void`\>

will handle running validations, error catching and state management.
